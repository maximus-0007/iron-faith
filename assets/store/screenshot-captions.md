# Screenshot Requirements

## iOS (App Store Connect)

**iPhone:**
- 6.7" (iPhone 14 Pro Max, 15 Pro Max): 1290 x 2796 px
- 6.5" (iPhone 11 Pro Max, XS Max): 1242 x 2688 px
- 5.5" (iPhone 8 Plus): 1242 x 2208 px

**iPad Pro:**
- 12.9" (6th gen): 2048 x 2732 px
- 11" (4th gen): 1668 x 2388 px

**Required:** 3-10 screenshots per device size

## Android (Google Play Console)

**Phone:**
- Minimum: 1080 x 1920 px (16:9)
- Maximum: 3840 x 2160 px

**7-inch Tablet:**
- 1080 x 1920 px

**10-inch Tablet:**
- 1200 x 1920 px

**Required:** 2-8 screenshots per form factor

# Tips for Creating Screenshots

1. **Show real value:** Display actual AI responses with Scripture references
2. **Highlight personalization:** Show the settings screen with options
3. **Demonstrate features:** Conversation list, chat interface, dark mode
4. **Keep text readable:** Ensure all text is legible at thumbnail size
5. **Use device frames:** Consider adding device frames for professional look
6. **Show both themes:** Include at least one dark mode screenshot
7. **Avoid clutter:** Keep screenshots clean and focused
8. **Real content:** Use authentic, helpful example conversations
